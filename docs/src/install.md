# Installation

FusionRouter is retrievable via [Wally](https://wally.run/), a package manager.

```toml
[package]
name = "user/project"
version = "0.1.0"
registry = "https://github.com/UpliftGames/wally-index"
realm = "shared"

[dependencies]
Fusion = "elttob/fusion@0.1.1-beta"
Router = "7kayoh/fusionrouter@1.0.0"
```
